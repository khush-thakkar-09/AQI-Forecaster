# Kolkata AQI Forecaster API
This Hugging Face Space hosts the FastAPI backend server for the Kolkata AQI Forecaster. It fetches weather and air quality data in real-time, runs it through the preprocessing pipeline, and predicts the 24-hour ahead AQI using a custom CNN-ILSTM deep learning model.
